General Information

This sample website showcases the website style style-1024px/. Most website styles include dynamic parts by default, for example, automatically generated menus and breadcrumb trails. All such dynamic parts are active in this showcase like they would be on a normal website.

How to use this style

This website style can be used when creating a new website by using the following command:

webgen create -b default -b style-1024px/ SITE_DIR

Or it can be applied later on to an already existing website by using the following command:

webgen apply style-1024px/
