Webgen::Tag::Sitemap

Summary

The following tag names are registered for this tag class: sitemap

This tag uses the following options:

  • common.sitemap.honor_in_menu
  • common.sitemap.any_lang
  • common.sitemap.used_kinds

Description

This tag is used to display a site map of the current website. The used nodes can be customized by setting the above mentioned configuration options accordingly.
